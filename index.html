National Park Website and Trailblazer App

Technologies used: ASP.NET, C#, Bootstrap, JQuery, HTML, CSS, Unity, Vuforia, Photoshop

A partner and myself were given the class project of designing a website that stores information about National Parks. The use cases were that each National Park would have a detail page which displays information about the park and it’s upcoming 5 day weather forecast. On top of creating a dynamic, responsive website, we created a mobile companion application that we named Trailblazer. Each park would have a number of markers at the park’s points of interest. When the user scans one of these markers with their phone, the app displays to the user their current location, interesting facts about the location, nearby points of interest, and accomodations, using augmented reality.



Birthday Card AR App

Technologies used: Unity, Vuforia, C#

This project came about when I was picking out a card for a friend’s birthday. As I was muttering about having to pay $5 dollars for a fancy piece of paper, I realized that greeting cards would be a great medium for augmented reality. When the user points their phone at the card, it displays 3D, augmented reality, content and plays the “Happy Birthday” song. I am currently working on expanding the app to include a web application that the person giving the card can use to add a personal message. Using SQL, the message would display when the user opens their card.
